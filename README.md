🧬 DNA Variant Detector

A Python-based machine learning application for detecting mutation-like patterns in DNA sequences using a TensorFlow Lite model.

🚀 Features
Accepts DNA sequences containing A, C, G, and T nucleotides
Converts DNA sequences into one-hot encoded representations
Uses a TensorFlow Lite model for prediction
Predicts whether a sequence is likely to contain mutation-like patterns
Displays prediction confidence
Calculates GC Content
Shows nucleotide distribution (A, C, G, and T)

Technologies Used- Python, NumPy, TensorFlow Lite, Pandas


Workflow
DNA Sequence Input
→ Input Validation
→ One-Hot Encoding
→ TensorFlow Lite Prediction
→ Confidence Score Generation
→ GC Content Analysis
→ Nucleotide Distribution

Example Input
ACGTACGTACGTACGTACGTACGTACGTACGTACGTACGTACGTACGTAC
